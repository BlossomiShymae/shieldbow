Readonly _rawTooltip

_rawTooltip: string
  • Defined in src/structures/SummonerSpell.ts:35

The raw tooltip of this summoner spell. This contains some HTML-like tags to help view this better on webpages.

Even though, these are supposed to be more detailed than the description. It is not recommended using either this OR tooltip. This is because they contain placeholders without values to fill them with. Use description instead.

See tooltip to view this with the HTML-like tags stripped out.
