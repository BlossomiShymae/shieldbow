## round(value) function

Rounds the value to up to 2 decimal points.

**Signature:**

```ts
export declare function round(value: number): number;
```

### Parameters

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| value | number | The value that needs to be rounded off. |


**Return type :** number
