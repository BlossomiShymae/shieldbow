Class Client

The league.ts client that enables you to interact with Riot Games' League of Legends API. Also connects to the Data Dragon + Community Dragon CDNs.

example

Here is how to use the client:

const myClient = new Client();
client.initialize('euw').then(() => {
  // All your code goes here.
});
